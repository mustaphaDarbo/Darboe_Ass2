Introduction

JavaScript is a powerful programming language used to create interactive effects within web browsers. It is an essential part of web development, allowing for dynamic content and engaging user experiences.

Variables

Variables in JavaScript are containers for storing data values. They can be declared using var, let, or const.

  • var - Declares a variable with function scope.
  • let - Declares a block-scoped variable.
  • const - Declares a constant value that cannot be reassigned.

Functions

Functions allow you to encapsulate reusable code. You can define a function using the function keyword followed by the function name.


function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Mustapha"));

Loops

Loops are used to repeat a block of code multiple times. JavaScript offers several types of loops, including:

  • for - Repeats a block of code a set number of times.
  • while - Repeats a block of code while a condition is true.
  • do...while - Executes a block of code once, then repeats it while a condition is true.

Objects

Objects are collections of properties, which are key-value pairs. You can create an object using curly braces {} and access its properties using dot or bracket notation.


let person = {
  name: "John",
  age: 30,
  greet: function() {
    return "Hello " + this.name;
  }
};
console.log(person.greet());
