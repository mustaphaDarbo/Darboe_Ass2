Introduction

JavaScript is a powerful programming language used to create interactive effects within web browsers. It is an essential part of web development, allowing for dynamic content and engaging user experiences.

Variables

Variables in JavaScript are containers for storing data values. They can be declared using var, let, or const.

  • var - Declares a variable with function scope.
  • let - Declares a block-scoped variable.
  • const - Declares a constant value that cannot be reassigned.

Functions

Functions allow you to encapsulate reusable code. You can define a function using the function keyword followed by the function name.


function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Mustapha"));

Loops

Loops are used to repeat a block of code multiple times. JavaScript offers several types of loops, including:

  • for - Repeats a block of code a set number of times.
  • while - Repeats a block of code while a condition is true.
  • do...while - Executes a block of code once, then repeats it while a condition is true.

Objects

Objects are collections of properties, which are key-value pairs. You can create an object using curly braces {}and access its properties using dot or bracket notation.


let person = {
  name: "John",
  age: 30,
  greet: function() {
    return "Hello " + this.name;
  }
};
console.log(person.greet());

Arrays

Arrays are used to store multiple values in a single variable. They are ordered lists, with values accessed by their index.


let fruits = ["apple", "banana", "cherry"];
console.log(fruits[1]); // Output: banana

Events

Events allow you to trigger actions in response to user interactions, like clicks, form submissions, and key presses. Event listeners are added using JavaScript.


document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

Error Handling

JavaScript provides ways to handle errors gracefully using try, catch, and finally blocks.


try {
  let result = riskyFunction();
} catch (error) {
  console.log("An error occurred: " + error.message);
} finally {
  console.log("This will run regardless of an error.");
}

Asynchronous JavaScript

Asynchronous JavaScript allows you to perform tasks in the background without blocking the main thread. This can be achieved using callbacks, promises, or async/await syntax.


async function fetchData() {
  let response = await fetch('https://api.example.com/data');
  let data = await response.json();
  console.log(data);
}
fetchData();
